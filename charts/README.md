# Charts Directory

This directory contains all visualization charts generated from the Speech Emotion Recognition project.

## Charts Included

### Data Distribution Charts (01-05)
1. **01_emotion_distribution.png** - Overall emotion distribution across all datasets
2. **02_dataset_distribution_pie.png** - Dataset source distribution (pie chart)
3. **03_emotion_by_dataset_stacked.png** - Stacked bar chart of emotions by dataset
4. **04_emotion_dataset_heatmap.png** - Heatmap of emotion-dataset relationships
5. **05_emotion_by_dataset_grouped.png** - Grouped bar chart of emotions by dataset

### Training Charts (06)
6. **06_training_history.png** - Model accuracy and loss over training epochs

### Performance Charts (07-11)
7. **07_confusion_matrix.png** - Confusion matrix of predictions
8. **08_per_class_accuracy.png** - Accuracy breakdown by emotion class
9. **09_confusion_matrix_normalized.png** - Normalized confusion matrix (percentages)
10. **10_precision_recall_f1.png** - Precision, recall, and F1-score comparison
11. **11_test_set_distribution.png** - Test set sample distribution

### Audio Sample Visualizations (12+)
12. **12_audio_sample_*.png** - Individual audio file analyses with waveforms, spectrograms, and predictions

## Usage

All charts are saved in high resolution (300 DPI) PNG format and can be used in presentations, reports, or publications.

Generated by: Speech Emotion Recognition Jupyter Notebook
