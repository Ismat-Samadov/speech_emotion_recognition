# Outputs Directory

This directory contains all data outputs, metrics, and reports from the Speech Emotion Recognition project.

## Files Included

### Data Distribution
- **emotion_distribution.csv** - Count of samples per emotion
- **dataset_distribution.csv** - Count of samples per dataset source

### Model Performance
- **classification_report.csv** - Detailed classification metrics (precision, recall, F1-score)
- **test_results.json** - Summary of test performance (accuracy, loss, sample counts)
- **confusion_matrix.csv** - Confusion matrix in tabular format
- **per_class_accuracy.csv** - Accuracy for each emotion class

### Training Metrics
- **training_history.csv** - Training and validation metrics per epoch

### Summary
- **file_summary.txt** - Summary of all generated files

## File Formats

- **CSV files** - Can be opened in Excel, Google Sheets, or pandas
- **JSON files** - Structured data, can be loaded programmatically
- **TXT files** - Plain text summaries

## Usage

These files can be used for:
- Further analysis in Excel/Python
- Creating custom visualizations
- Model performance reporting
- Academic papers and presentations

Generated by: Speech Emotion Recognition Jupyter Notebook
